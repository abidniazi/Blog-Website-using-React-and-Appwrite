# ToolForge — Free Online Tools Platform
 
**Live site: [freetoolforge.org](https://freetoolforge.org)**
 
A free online toolkit with 38 tools across 5 categories. No signup required. No advertisements. All file processing happens locally in your browser — files are never uploaded to any server.

## Tech Stack
- **Framework**: Next.js 14 App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel
- **PDF processing**: pdf-lib, pdfjs-dist
- **AI**: Google Gemini API (gemini-2.0-flash-lite)
- **Compiler**: Judge0 CE
## Privacy
All file processing runs locally in the browser. No files are uploaded to any server. No data is stored beyond standard Google Analytics page tracking.
